# ADR 0042: GitHub collaboration templates

## Status

Accepted

## Context

The project now has CI, coverage reporting, release documentation, CLI workflows, and a growing set of modules.

Future issues and pull requests should follow a consistent structure so bugs, features, and reviews are easier to triage.

## Decision

Add GitHub issue and pull request templates.

The templates include:

- Bug report template
- Feature request template
- Pull request template
- Issue template configuration

The pull request template includes the local quality commands expected before merging.

## Consequences

Issues become easier to reproduce and prioritize.

Pull requests have a clearer definition of done.

The project has a stronger collaboration workflow for future contributors.
