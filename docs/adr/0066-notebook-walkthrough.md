# ADR 0066: Advanced demo notebook walkthrough

## Status

Accepted

## Context

The project now includes a one-command advanced demo workflow and a professional README.

However, reviewers may still benefit from a guided notebook that shows the end-to-end research workflow in a readable format.

A notebook is useful for portfolio review because it combines narrative explanation, code, and generated outputs.

## Decision

Add an advanced demo walkthrough notebook.

The notebook demonstrates:

- The core research question
- The advanced demo workflow
- Generated input files
- Exported research tables
- Advanced memo preview
- Interpretation guidance

## Consequences

The project is easier to review interactively.

A recruiter, professor, or investment reviewer can open the notebook and quickly understand the full research pipeline.

Future work can add notebooks using real historical data and rendered charts.
