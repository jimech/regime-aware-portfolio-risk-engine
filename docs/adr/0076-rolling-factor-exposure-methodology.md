Accepted

## Context

Rolling factor exposure analysis is now available in the research engine and integrated into the advanced research package.

However, reviewers and users need a clear methodology note explaining how rolling betas are estimated, what the exported fields mean, and how the analysis supports the project thesis.

## Decision

Add a dedicated rolling factor exposure methodology document.

The document explains:

- Required inputs

- Rolling regression structure

- Interpretation of alpha, beta, R-squared, residual volatility, and dominant factor

- Relationship to the regime-aware portfolio thesis

- Key limitations

## Consequences

The rolling factor exposure feature is easier to review, audit, and interpret.

The project now has stronger documentation connecting the statistical output to the investment research question.
