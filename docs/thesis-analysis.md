# Thesis Analysis: Regime-Aware Portfolio Risk Engine

## Research Question

Can changing market regimes be detected from financial time-series data, and can portfolio risk exposure be adjusted dynamically in response?

## Regime Detection

The project uses rolling return, volatility, trend, momentum, and correlation features to detect changing market environments.

## Dynamic Allocation

The project compares a static benchmark portfolio against a regime-aware dynamic allocation process.

## Conclusion

This project demonstrates a regime-aware portfolio research workflow that connects market-state detection, risk analytics, allocation, validation, and investment reporting.
